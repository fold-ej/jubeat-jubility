jubeat Jubility Visualizer

유빌리티 계산 + 시각화 스크립트

복사 완료
사용법
1. e-amusement 플레이 데이터 페이지에 로그인
2. 개발자 도구(F12) → Console
3. 위 버튼으로 복사한 코드 붙여넣기
4. 약 2~4분 후 PNG 이미지 자동 다운로드
使い方
1. e-amusementプレーデータにログイン
2. デベロッパーツール(F12) → Console
3. 上ボタンでコピーしたコードを貼り付け
4. 約2〜4分後PNG画像が自動的にダウンロード
모바일에서 사용하기 / モバイルでの使用
북마클릿(bookmarklet)을 이용하면 핸드폰에서도 실행할 수 있습니다.
iOS (Safari) 1. 아무 페이지나 북마크에 추가
2. 북마크 편집 → URL을 아래 코드로 교체
javascript:void(document.head.appendChild(Object.assign(document.createElement('script'),{src:'https://fold-ej.github.io/jubeat-jubility/jubility.js'}))) 3. e-amusement 플레이 데이터에 로그인
4. 주소창 탭 → 저장한 북마크 선택
5. 약 2~4분 후 PNG 이미지 자동 다운로드
Android (Chrome) 1. 아무 페이지나 북마크에 추가
2. 북마크 편집 → URL을 아래 코드로 교체
javascript:void(document.head.appendChild(Object.assign(document.createElement('script'),{src:'https://fold-ej.github.io/jubeat-jubility/jubility.js'}))) 3. e-amusement 플레이 데이터에 로그인
4. 주소창에 북마크 이름 입력 → 북마크 선택
5. 약 2~4분 후 PNG 이미지 자동 다운로드
ブックマーク(bookmarklet)を使えばスマホでも実行できます。
iOS (Safari) 1. 適当なページをブックマークに追加
2. ブックマークを編集 → URLを以下のコードに置換
javascript:void(document.head.appendChild(Object.assign(document.createElement('script'),{src:'https://fold-ej.github.io/jubeat-jubility/jubility.js'}))) 3. e-amusement プレーデータにログイン
4. アドレスバーをタップ → 保存したブックマークを選択
5. 約2〜4分後PNG画像が自動的にダウンロード
Android (Chrome) 1. 適当なページをブックマークに追加
2. ブックマークを編集 → URLを以下のコードに置換
javascript:void(document.head.appendChild(Object.assign(document.createElement('script'),{src:'https://fold-ej.github.io/jubeat-jubility/jubility.js'}))) 3. e-amusement プレーデータにログイン
4. アドレスバーにブックマーク名を入力 → ブックマークを選択
